# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Hymoex, please report it responsibly.

**Do not open a public issue.** Instead, send an email to:

**joel@pymut.ai**

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fix (if applicable)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 1 week
- **Fix or mitigation:** as soon as reasonably possible, depending on severity

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Disclosure Policy

We follow coordinated disclosure. We will work with you to understand and address the issue before any public disclosure. Credit will be given to reporters unless they prefer to remain anonymous.
