## Welcome to Hymoex

**Hymoex** (Hybrid Modular Coordinated Experts) is an architectural paradigm for building scalable multi-agent systems with Mixture-of-Experts gating, progressive migration, and framework-agnostic patterns.

## Key Features

- **7-role agent taxonomy** — Manager, Integrator, Expert Manager, Supervisor, Expert, Executor, Perceiver
- **3 deployment modalities** — One-Line MoE, One-Line Supervisor, MoE MultiLine
- **MoE gating** — 96.7% expert selection accuracy via LLM-based routing
- **Progressive migration** — Scale from M1 to M3 preserving 100% of existing agents
- **7 framework examples** — Pydantic AI, LangGraph, CrewAI, AutoGen, Swarm, Vercel AI SDK, Mastra
